低功耗无线温度节点

使用 MSP430 和 CC1101 实现的低功耗温度节点

View on GitHub Download .zip Download .tar.gz

介绍

本项目是我的毕业设计，使用MSP430和CC1101实现冷藏车厢内温度的采集，温度采集节点被放置于冷藏车厢内顶部四个角。 整个设计包括硬件设计和程序设计，硬件设计使用PCB软件Altium Designer Summer 09设计完成，程序设计使用IAR for MSP430编程环境实现。
